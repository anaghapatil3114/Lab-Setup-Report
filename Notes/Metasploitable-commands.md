# Metasploitable Commands

## 1. Check IP Address
ifconfig
# Shows IP address of Metasploitable.

## 2. Check Active Services
netstat -tuln
# Lists open TCP/UDP ports.

## 3. Test Connectivity
ping <Kali_IP>
# Example:
ping 192.168.56.100

## 4. Network Troubleshooting
route -n
# Shows routing table.
